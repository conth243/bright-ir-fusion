# Bright IR Fusion

在PC上运行的一个调试双光融合效果的软件，双光是指可见光图像和红外热成像。

## 项目描述

Bright IR Fusion是一个基于C++和OpenCV开发的UVC摄像头视频流显示软件，支持摄像头检测、视频流捕获和显示、UI界面交互等功能。

## 功能特性

- **摄像头检测**：自动枚举和检测可用的摄像头设备
- **视频流显示**：支持实时视频流捕获和显示
- **分辨率管理**：支持不同分辨率的设置
- **UI界面**：提供摄像头选择对话框和控制按钮
- **错误处理**：添加了异常处理机制

## 技术栈

- **C++11**：核心代码语言
- **OpenCV**：图像处理和视频捕获
- **CMake**：构建系统
- **MSVC**：编译器
- **Windows SDK**：系统API支持
- **Git**：版本控制

## 构建和运行

### 构建环境

- **操作系统**：Windows 11
- **编译器**：MSVC (Visual Studio Build Tools 2026)
- **构建工具**：CMake 4.3.0
- **依赖库**：OpenCV 4.12.0

### 构建步骤

1. 克隆项目代码
2. 配置CMakeLists.txt中的OpenCV路径
3. 运行CMake生成构建文件
4. 使用MSVC编译项目

### 运行

- 双击生成的main.exe可执行文件
- 选择可用的摄像头设备
- 查看实时视频流
- 使用界面上的按钮进行控制

## 项目结构

- **include/**：头文件目录
- **src/**：源代码目录
- **docs/**：文档目录
- **main.exe**：编译生成的可执行文件
- **test_report.md**：测试报告
